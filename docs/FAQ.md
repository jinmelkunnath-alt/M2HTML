# FAQ

## Does MarkForge require internet access?

No. The C library, CLI, website, playground and WebAssembly module are designed for offline operation.

## Is the website backed by a server?

No. The GitHub Pages site is static and the playground executes locally in the browser.

## Which renderers are supported?

HTML, JSON, XML, plain text and normalized Markdown formatting.

## Can plugins be installed from the internet?

No. Plugin management is local-only by design.

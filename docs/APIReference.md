# API Reference

The public C API is exposed through `include/markforge/markforge.h`. This alpha API is intended for evaluation and may evolve before a stable 1.0 release.

## Implemented API Families

- Memory, containers, strings, filesystem, logging and diagnostics.
- Tokenizer and token stream.
- Parser and AST.
- HTML, JSON, XML, plain text and simple Markdown formatter renderers.
- Renderer manager, hooks, flat configuration, static plugin metadata, themes and CLI commands.
- Developer tooling APIs for linting, validation, profiling, benchmark reports, inspection, watch smoke mode and preview server smoke mode.

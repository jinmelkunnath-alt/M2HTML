# Versioning Policy

MarkForge currently uses alpha versioning. The current version is `0.5.0-alpha`.

## Alpha Policy

- APIs are intended to be kept compatible where practical.
- Correctness and memory safety fixes may change behavior before 1.0.
- Documentation must distinguish implemented, partial and planned functionality.

## Future Stable Policy

A future 1.0 release should adopt Semantic Versioning with patch, minor and major compatibility rules after the parser, renderer and public API contracts are validated by stronger tests.

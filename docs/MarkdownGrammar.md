# Markdown Grammar

MarkForge alpha implements a limited Markdown subset. It is not CommonMark-complete.

## Implemented Subset

- ATX headings.
- Paragraph lines.
- Simple unordered and ordered list lines.
- Blockquote lines.
- Horizontal rules.
- Fenced code blocks.
- Basic inline code, bold, italic, strike, highlight, links, images and inline math.

## Not Implemented

Full CommonMark block/inline grammar, nested list semantics, full tables, task lists, footnotes, definitions, automatic links/emails and full HTML block grammar.

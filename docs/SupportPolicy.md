# Support Policy

MarkForge `0.5.0-alpha` is supported for evaluation, testing and contribution. It is not recommended for production use with untrusted documents.

## Target Platforms

The native C code targets Linux, macOS, Windows and WSL. Some alpha features have platform limitations; for example, the preview server implementation is POSIX-only in the current source.

## Reporting Issues

Please include platform, compiler, command line, input document and observed output.

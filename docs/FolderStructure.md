# Folder Structure

- `src/`: C implementation files.
- `include/markforge/`: public headers.
- `docs/`: architecture and developer documentation.
- `configs/`: sample and default configuration files.
- `tests/`: unit tests and test runner.
- `benchmarks/`: benchmark runner.
- `examples/`: usage examples.
- `plugins/`: future plugin integration surface.
- `website/`: future documentation website.
- `assets/`: static project assets.
- `themes/`: future renderer themes.
- `tools/`: developer tools.
- `scripts/`: automation scripts.
- `build/`: generated build output.
- `.github/`: issue templates, PR template, and CI.
